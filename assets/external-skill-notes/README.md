# External Skill Notes

This folder stores paraphrased notes from external and local meta skills.

The notes are not raw copies. They are distilled patterns to help improve this workspace later.

## Files

- `wechat-writer.md`
- `doc-coauthoring.md`
- `article-review.md`
- `meta-skills.md`

## How to use

When upgrading this skill:

1. check whether the current problem has already been solved by one of these source patterns
2. adapt the pattern to research-report constraints
3. write the adapted rule into `references/`
4. add or update an eval if the pattern fixes a recurring failure
