# Benchmark Samples

This folder stores distilled notes about benchmark reports.

Do not paste entire reports here. Keep the focus on reusable patterns:

- structure
- evidence organization
- figure and table responsibilities
- valuation and financial wording boundaries
- chapter rhythm

Each benchmark note should answer:

1. why this sample is worth learning from
2. what to imitate
3. what not to copy
4. where the important sections live
