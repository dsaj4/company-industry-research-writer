# 风格样本

这个目录用于保存从真实券商研报中提炼出的风格锚点。

它们的作用不是替代 thesis、结构和证据规划，而是在这些已经成立之后，把成文质量往“像成熟研报”方向推。

## 当前样本集

- `gf-front-page-and-core-points.md`
- `gf-chapter-opening-patterns.md`
- `gf-argument-density-patterns.md`
- `gf-risk-writing-patterns.md`

## 来源规则

不要把原始研报的大段文字直接搬进来。

更好的做法是：

- 标明参考了哪份研报、哪些页面
- 提炼可复用的写法
- 把它转换成模板、检查项或迁移规则

## 使用方式

在起草或重写时：

1. 先锁定 thesis 和证据任务
2. 再选一份与当前章节最相关的风格样本
3. 模仿写作动作，不模仿具体公司内容

## 适合放进来的内容

- 首页核心观点的组织方式
- 章节开头的起句方式
- 高密度论证段的写法
- 风险提示的写法

## 不要放进来的内容

- 长篇受版权保护的原文
- 没有复用价值的公司事实
- 和真实研报写法无关的抽象写作建议
