# Thesis

## 中心判断

晶泰控股更适合被理解为一家 AI+机器人驱动的研发基础设施平台，而不是单一的 AI 制药工具公司。其稀缺性不只在算法能力，而在于把量子物理计算、AI 模型、机器人自动化和高价值客户场景组织成一条可迭代的研发闭环。

## 本次切片需要回答的 4 个问题

1. 为什么首页第一条要先定义公司 archetype，而不是先讲赛道空间？
2. AI4S 赛道的“值得看”，应该如何压缩成一句带证据支撑的判断？
3. 公司概况开篇最应该证明的，是“技术先进”，还是“平台化闭环已经形成”？
4. 哪些不确定性必须在首页阶段就提前交代，避免整段文字显得过满？

## 本次切片的 supporting judgments

### 1. 身份判断

公司不是单点能力商，而是把算法、实验和数据回流连接起来的平台型研发服务商。

### 2. 赛道判断

AI4S 仍处于早期渗透阶段，市场空间成立，但真正值得关注的是其对研发流程的重构能力。

### 3. 优势判断

公司的高价值在于“模型设计 + 机器人执行 + 实时反馈”协同后的飞轮，而不是任一单点模块。

### 4. 边界判断

商业化放量和跨行业复制仍需持续验证，因此首页必须把风险边界提前写进去。
